# Front-end da Aplicação Huggy

Este é o front-end da aplicação de captação de clientes com integração Huggy, construído com Vue.js.

## Configuração do Projeto

Siga as instruções abaixo para configurar e iniciar o projeto em seu ambiente local.

### Instalação de Dependências

Execute o comando abaixo para instalar as dependências necessárias do projeto:

```bash
npm install
```
